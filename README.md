# Smart-traffic-control-system
Traffic congestion is one of the biggest problems in modern cities. Traditional traffic signals work on fixed timers, which cannot adjust to changing traffic conditions. This often causes unnecessary delays, traffic jams, and longer waiting times. To solve this problem, we introduces a Smart Traffic System.
The Smart Traffic System is developed using C++  to simulate traffic signals and vehicle movement at a road intersection. It provides a graphical interface and demonstrates important Operating System (OS) and Parallel and Distributed Computing (PADC) concepts such as multithreading, synchronization, and parallel execution.
